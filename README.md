# 贰伴 — 微信公众号源代码编辑器

> 为中科院物理所微信公众号值日小编打造的微信公众号源代码编辑器

## 核心功能

- **HTML 源代码编辑** — 直接查看和编辑微信公众平台文章 HTML
- **实时预览** — 右侧分栏即时渲染 HTML 效果
- **安全过滤** — 自动清理微信不支持的标签和样式

---

## 安装步骤

### 1. 打开扩展管理页面

在浏览器地址栏输入：
- **Edge**: `edge://extensions`
- **Chrome**: `chrome://extensions`

### 2. 开启开发者模式

页面左下角（Edge）或右上角（Chrome），打开 **「开发人员模式」** / **「开发者模式」** 开关。

### 3. 加载插件

1. 点击 **「加载解压缩的扩展」**（Edge）或 **「加载已解压的扩展程序」**（Chrome）
2. 选择 `Erban-source-editor` 文件夹
3. 确认

## 使用方法

1. 打开微信公众号文章编辑页面
2. 工具栏中（文章末尾）会出现绿色 **「贰伴 · 源代码」** 按钮
3. 点击按钮打开编辑器
